# AE2 Personal ME System

An AE2 addon that provides a wearable personal terminal that replaces your inventory key

## Features

- **Personal Wireless Terminal**: A wireless terminal worn as a Curio
- **Inventory Key Override**: Press E to open the ME terminal instead of vanilla inventory
- **Always Connected**: Infinite range and no power consumption when bound
- **Bind from WAP**: Bind your terminal directly from the Wireless Access Point GUI with a button
- **FancyMenu Compatibility**: When FancyMenu is installed, AE2 screens become customizable (removes blacklist restriction)(mixin)

## Usage

1. Obtain a Personal Wireless Terminal
2. Equip it in a Curios slot (or keep in inventory)
3. Open a Wireless Access Point and click the bind button in the left toolbar
4. Press E - your ME terminal opens instead of vanilla inventory
